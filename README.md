# 📖 About the Project

**Image Processing Toolkit** is a desktop-based GUI application developed using **Python**, **NumPy**, **OpenCV**, **Pillow**, and **CustomTkinter**.

The objective of this project was to understand the fundamentals of **digital image processing** by implementing common image manipulation techniques in a user-friendly graphical interface instead of relying on professional image editing software.

Users can load an image, apply multiple filters and transformations in real time, crop selected regions interactively, and save the processed image.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Image-Processing-Toolkit.git
```

## Navigate to Project

```bash
cd Image-Processing-Toolkit
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```
